# Internal links

<!-- does not need translation -->

To link to the right internal guides, references, and endpoints, you can use the
following syntax.

```json
[Get a file by ID](endpoint://get_files_id)
[A file](resource://file)
[Get a file](guide://files/get)
[Box](https://box.com)
```
