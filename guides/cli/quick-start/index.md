# OAuth 2.0を使用したBox CLIの使用

<!--alex ignore executed-->

Box CLIは、使い勝手の良いコマンドラインツールです。これにより、開発者でも開発者以外のユーザーでもBox APIを利用してルーチンや一括操作を実行できるようになります。コードを書く必要はありません。これらの操作は、一連の[コマンド][commands]によって実行されます。
