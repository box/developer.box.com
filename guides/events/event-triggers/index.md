# イベントトリガー

イベントは、ユーザー、項目、またはその他のリソースによってコミットされたアクションに基づいてイベントストリームに表示されます。[`GET /events`](e://get_events)エンドポイントのレスポンスは、イベントをトリガーしたイベントによって変わります。
