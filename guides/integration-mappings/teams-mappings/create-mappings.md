# Teams統合マッピングの作成

`POST integration_mappings_teams`呼び出しを使用してマッピングを作成します。この呼び出しを動作させるには、`box_item`パラメータと`partner_item`パラメータが必要です。これらのパラメータはそれぞれ、BoxフォルダとTeamsチャネルを示します。
